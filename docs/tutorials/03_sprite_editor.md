# Tutorial 3: Draw and Convert a Sprite

Goal: create a sprite and convert it to a C array.

## 1. Open Sprite Lab

Click **Sprite Lab** in the navigation bar.

## 2. Create a new sprite

1. Click **New Sprite**.
2. Name it `player`.
3. Use 16 by 16 pixels.
4. Pick a paint color.
5. Draw a simple character.

## 3. Save

Click **Save**. The sprite is stored as JSON in the backend database.

## 4. Convert to C

Click **Convert to C**. The C panel shows a `uint16_t` RGB565 array.

## 5. Find the artifact

Open **Artifacts**. Download the generated sprite C file.
